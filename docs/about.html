About

A VS Code theme built around two simple ideas.

What alpha is

alpha is a free, open-source theme for Visual Studio Code. It pairs GitHub's official interface palette with a perceptually balanced syntax palette. The window chrome, sidebar, editor surfaces, terminal, and git decorations are matched byte-for-byte to GitHub Dark Default and GitHub Light Default v6.3.5. The code colors are designed so no single token type visually overpowers another.

Why two variants

The dark variant locks every syntax color at HSL(h, 100%, 68%) — strict isoluminance, identical perceived weight on a dark background. The light variant retunes lightness per hue so every color clears WCAG-AA contrast on white while preserving the same seven hue identities. Same intent, adapted to the medium.

Why GitHub's UI

GitHub Dark and GitHub Light are among the most carefully tuned editor interfaces shipping today — readable for long sessions, accessible, and visually quiet. There was no reason to reinvent that. alpha keeps the chrome you already know and spends its design budget where it matters: the colors of the code itself.

The palette

Seven hues, one per major token family — variables, constants, classes, strings, functions, operators, keywords. The hue assignments are warm-clustered: red / orange / yellow for the three highest-frequency tokens (variables, constants, classes), leaving cool tones for the rest. This is deliberate — most code is variables and types, and the eye discriminates warm hues more easily.
