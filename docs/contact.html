Bug reports

For UI mismatches, color quirks, or anything that looks off, use the bug report template on GitHub. Including your VS Code version, the alpha version, and a screenshot covers most of what's needed to reproduce.

Feature & color requests

Have an idea for a tweak, a new variant, or a hue that would fit the palette better? Open a feature request. Color suggestions are welcome — send the hex value and a short note on why it would slot in.
